How to play Memory Game?

  • On start 4 images will be display on screen.
  • Player should remember the images.
  • After 5 sec, the images will disappear.
  • Multiple images will display on screen,player must select 4 images that match previously displayed images.
  • After selecting all 4 images, program will display result.
  • Program will also display game score.
  • Player can continue by clicking start button.
